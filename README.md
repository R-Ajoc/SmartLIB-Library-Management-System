# SmartLIB - Library Management System

A web-based Library Management System with **four user roles**: Student, Teacher, Librarian, and Staff.  
Built with **HTML, CSS, JavaScript** (frontend) and **PHP OOP** (backend).

---

## Documentation

For guidance and technical details, refer to the following included markdown files:

- **SYSTEM_DOCS.md** – Complete system documentation (architecture, database, code structure, and design decisions).  
- **USER_GUIDE.md** – Instructions for users on how to navigate and use the system.

These files are included in the repository alongside the project source code.
